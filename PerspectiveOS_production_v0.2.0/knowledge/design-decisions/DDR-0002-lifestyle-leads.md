# DDR-0002: Lifestyle führt, Hilfsmittel begleiten

## Entscheidung
Content beginnt bevorzugt mit Lifestyle: Outfit, Kaffee, Reise, Alltag, Dating, Shopping, Arbeit.

## Begründung
So wird die Person nicht auf Hilfsmittel reduziert.

## Konsequenz
Bei Post-Briefings wird immer zuerst die Alltagssituation definiert, danach das Hilfsmittel.
