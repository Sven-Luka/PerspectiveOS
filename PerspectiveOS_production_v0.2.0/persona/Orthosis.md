# Orthosis Guide

## Pflicht
- Orthese am linken Bein der Person
- von vorne betrachtet erscheint sie rechts im Bild
- Bauerfeind-ähnliche Knieorthese
- schwarzer/dunkelblauer textiler Körper
- seitliche Gelenke
- breite Klettgurte
- glaubwürdige medizinische Proportionen
- sitzt über Strumpfhose oder Leggings

## Nicht erlaubt
- Fantasie-Orthesen
- zu dünne modische Schiene
- futuristische Exoskelette
- wechselndes Bein
- fehlende Gelenke
- zu perfekte Studio-Darstellung

## Darstellung
Die Orthese darf sichtbar sein, aber nicht in jedem Bild das Hauptmotiv. Sie gehört einfach dazu.
