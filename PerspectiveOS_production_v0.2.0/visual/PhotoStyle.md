# Photo Style

## Grundstil
Casual Documentary.

## Bevorzugte Perspektiven
- Selfie von oben auf Beine/Orthese
- POV im Zug
- Spiegelaufnahme ohne Gesicht
- von hinten beim Gehen
- Alltagsschnappschuss
- leicht bewegte Bilder

## Orte
- Zug
- Bahnhof
- Café
- Supermarkt
- Wohnung
- Aufzug
- Parkbank
- Shoppingcenter
- Straße

## Wichtig
Die Bilder sollen natürlich wirken, nicht wie perfekte KI-Renderings.
