# Codex Brief

## Ziel
Dieses Repository soll als Arbeitsgrundlage für Perspective OS dienen.

## Aufgabe für Codex
1. Repo-Struktur prüfen.
2. Markdown-Linting vorbereiten.
3. Optional ein Python-Skript erstellen, das neue Post-Briefings aus einem Template erzeugt.
4. Optional ein Asset-Index-Skript erstellen, das Bilder im assets-Ordner katalogisiert.
5. Keine kreativen Inhalte eigenständig verändern, ohne Briefing.

## Rollenverteilung
- ChatGPT / Creative Director: Marke, Sprache, Strategie, Content.
- Codex / Technical Director: Struktur, Skripte, Automatisierung, Repo-Pflege.
