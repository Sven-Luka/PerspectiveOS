# Perspective OS

> **Sichtbar sein dürfen. Perspektivwechsel statt Vorurteile.**

Perspective OS ist das Brand- und Content-Operating-System hinter einem Lifestyle- und Inklusionskanal.

Der Kanal zeigt nicht primär Hilfsmittel.  
Der Kanal zeigt einen Menschen, zu dessen Leben Hilfsmittel selbstverständlich dazugehören.

## Nordstern

> Wir möchten nicht, dass Menschen nach unseren Beiträgen nur mehr über Behinderung wissen.  
> Wir möchten, dass sie Menschen mit Behinderung anders sehen.

## Markenformel

```text
Perspective =
Lifestyle
+ Mode
+ Humor
+ Perspektivwechsel
+ Selbstverständliche Sichtbarkeit
- Mitleid
- Schock
- Verstecken
- Perfektion
```

## Produktionsstandard

- Feed-Format: **1080 × 1350 px / 4:5**
- Bildstil: **Smartphone, natürlich, nicht zu perfekt**
- Layout: **luftig, wenig Text, viel Bild**
- Regel: **Lifestyle führt. Hilfsmittel begleiten.**
