# Perspective OS UI Kit References

These assets define deterministic layout and overlay direction for Perspective Studio.

Use these files for composition, typography, color, icon, badge, and CTA guidance.
Do not send layout reference screenshots as image-generation inputs, because models may copy visible text, stickers, badges, or UI bars into the generated photo.

## Imported Files

- `layout_carousel_overview_2026-06-25.png`: carousel overview reference for rhythm, badges, icon row, CTA slide, and brand accents.
- `layout_slide_01_2026-06-25.png`: single-slide reference for hook layout, highlight block, bottom icon row, counter, and branding.
- `icons/`: source icon set from `PerspectiveOS_UI_Kit_v1.zip`.
- `styles/STYLE_GUIDE.md`: color and typography rules.
- `templates/CAROUSEL_TEMPLATE.md`: carousel structure.

## Production Rule

The image model generates a clean photo only. Perspective Studio applies UI text, labels, icons, counters, badges, highlights, and branding afterward through a deterministic layout composer.

## Current Master Direction

The approved second-generation carousel family lives in `../master/carousel_v2/`. It is the visual QA reference for hook, icon-tip, explanation, and photo-led CTA slides.
