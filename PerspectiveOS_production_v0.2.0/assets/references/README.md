# Reference Library v0.1

Diese Bibliothek enthält die Original-Referenzen für Perspective OS.

## Kategorien
- `character/`: Körperbau, Silhouette, Haar-/Figurhinweise
- `orthosis/`: Bauerfeind-ähnliche Orthese, Ansichten und Details
- `orthopedic_shoes/`: Orthopädische Schuhe / Sneaker
- `outfits/`: Kleidung, Strumpfhosen, Kombinationen
- `diapers/`: Inkontinenzprodukte, Bund, Silhouette, Produktdetails
- `layouts/`: CI, Carousel-Layout, Typografie, Wiedererkennung
- `locations/`: Zug, Zuhause/Büro, Outdoor
- `rejected/`: generierte Negativbeispiele, nicht als Master nutzen

## Regel
Diese Bilder sind Referenzdaten. Sie sollen Konsistenz sichern – nicht automatisch veröffentlicht werden.
