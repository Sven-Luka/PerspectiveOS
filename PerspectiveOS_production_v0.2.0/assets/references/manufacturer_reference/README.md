# Manufacturer Product References

Hier gehören ausschließlich offizielle Herstellerbilder oder freigegebene Produktfotos hinein.

Aktuell wurden keine externen Herstellerfotos hinzugefügt, weil sie urheberrechtlich geschützt sein können.

Empfohlene Struktur:

```text
manufacturer_reference/
├── tena/
├── seni/
├── abena/
├── molicare/
└── attends/
```

Pro Bild bitte eine `.md`-Metadatei mit Quelle, Nutzungsstatus und Produktname anlegen.
